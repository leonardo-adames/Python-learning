## 🧠 Descripción General
Este repositorio documenta mi proceso de aprendizaje de Programación en Python, enfocándose en teoría, ejercicios prácticos y proyectos guiados. El objetivo es proveer un camino de estudio claro, completo y aplicable para cualquier persona que busque dominar conceptos esenciales del lenguaje desde cero.

<br>

## 🗂️ Estructura del Repositorio

**Este repositorio está organizado para optimizar la navegación pedagógica:**

```plain text
📁 /docs/        → Material teórico y recursos del curso
📁 /notes/       → Apuntes personales y explicaciones resumidas
📁 /exercises/   → Ejercicios prácticos con soluciones
📁 /projects/    → Proyectos reales y retos aplicados
📄 LICENSE       → Licencia de uso (MIT)
📄 README.md     → Documentación principal (este archivo)
```

Cada carpeta representa un nivel de aprendizaje, desde la teoría hasta la aplicación práctica.

## 🧾 Temario Principal

**Temas incluidos en este repositorio en la carpeta `docs`:**

```plain text
📌 00 - Introducción General

📌 01 - Python RoadMaps

📌 02 - Python-Core-to-Data-Analytics

📌 03 - Python-Core-to-Data-Science

📌 04 - Python-Core-to-Web-Development

(Esta lista puede crecer según avances en el curso. Cada tema contiene una estructura interna + una ruta de aprendizaje adaptada a cada nivel de conocimiento, area de interes o sector)
```

## 🚀 Cómo Empezar

**Clona el repositorio y comienza por la carpeta de teoría:**

```bash
git clone https://github.com/leonardo-adames/Python-learning.git
cd Python-learning
```

**Desde ahí, abre las carpetas:**

```plaun text
docs/ → teoría y recursos

exercises/ → ejercicios paso a paso

projects/ → proyectos reales para practicar y aplicar lo aprendido

(Puedes usar LeetCode y CodeWar, etc para reproducir los ejercicios.)
```
