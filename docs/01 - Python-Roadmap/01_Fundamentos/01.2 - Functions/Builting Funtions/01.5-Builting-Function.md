## 2.1.1 - Built-in Function en Python

Python incluye un amplio conjunto de **funciones integradas** (`built-in functions`), que son funciones predefinidas que puedes utilizar en cualquier momento sin necesidad de importarlas. Estas funciones proporcionan funcionalidades esenciales, desde operaciones matemáticas básicas hasta la manipulación de tipos de datos y la interacción con el sistema. 

A continuación se presenta una lista de algunas de las funciones integradas más comunes en Python:
