# **Verificar la instalación**

Una vez que la instalación haya finalizado, es conveniente verificar que Python se haya instalado correctamente en tu sistema.

Para ello, abrimos una ventana de terminal y escribimos el siguiente comando,

con el que verificaremos la versión de Python instalada:

```bash
python --version
```
<br>

Este comando debería imprimir la versión de Python instalada en tu sistema. Si ves algo como:

```bash
Python 3.11.0
```

Donde **`3.X.X`** es la versión de Python **¡felicidades!** Has instalado Python correctamente en tu sistema 🥳, Entonces Python se ha instalado correctamente.
