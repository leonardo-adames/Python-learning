# 📚 Alcance exacto de los ejercicios

Los ejercicios están diseñados exclusivamente para practicar:

`if`, `elif`, `else`

bucles `for` y `while`

operadores (`+ - * / % == != < > <= >=`)

strings (indexación, slicing, métodos básicos)

type casting (`int()`, `float()`, `str()`)

entrada de datos con `input()`

###❌ No se usa:

listas avanzadas`

diccionarios

funciones definidas por el usuario

librerías externas
