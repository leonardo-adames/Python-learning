<h1 align="center">📘 Python – Ejercicios de Manejo de Strings y Caracteres de Escape</h1>

Este repositorio contiene una colección de ejercicios prácticos de Python orientados al aprendizaje de los fundamentos del lenguaje, con énfasis en el manejo de cadenas de texto (strings), el uso de métodos nativos de string, caracteres de escape y estructuras de control básicas.

Los ejercicios están diseñados para estudiantes que se encuentran en una etapa inicial del aprendizaje, antes de introducir librerías externas como `pandas` o `numpy`, y se apoyan exclusivamente en funcionalidades incluidas en el núcleo del lenguaje Python.

<h1 laign="center">🎯 Objetivos de aprendizaje</h1>

A través de estos ejercicios, el estudiante podrá:

Manipular cadenas de texto utilizando métodos nativos de str

Comprender y aplicar caracteres de escape (`\n`, `\t`, `\"`, `\\`, `etc`.)

Utilizar condicionales (`if`, `elif`, `else`) para validar información

Recorrer strings mediante bucles for

Aplicar validaciones básicas con métodos como:

`.strip()`

`.lower()` / `.upper()`

`.replace()`

`.split()` / `.join()`

`.isdigit()`, `.isalpha()`, `.isalnum()`

Trabajar con entrada de datos usando `input()`

<h1 laign="center">🧩 Contenido de este apartado </h1>

Los ejercicios incluyen, entre otros temas:

Normalización de texto

Búsqueda y reemplazo de palabras

Validación de contenido textual

Separación y unión de cadenas

Impresión de texto formateado usando caracteres de escape

Manejo de comillas y rutas de archivos

Validaciones simples de correos electrónicos

Todos los ejercicios están planteados de forma progresiva, siguiendo una lógica similar a la utilizada en cursos introductorios formales de Python.

<h1>⚙️ Requisitos</h1>

Python 3.x

No se requieren librerías externas
